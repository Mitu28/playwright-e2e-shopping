# Playwright E2E Shopping Automation

This repository contains an **end-to-end automation test suite** for a sample e-commerce website using **Playwright**.  

The automation covers the following flow:  
- User login  
- Browse products  
- Add a specific product to cart  
- Checkout process  
- Order verification  

---

## 🛠️ Technologies Used

- **Playwright Test** - For browser automation and E2E testing  
- **Node.js** - JavaScript runtime  
- **dotenv** - For managing environment variables  
- **Chromium** - Default browser for testing
